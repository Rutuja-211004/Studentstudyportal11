Student Study Portal

📚 Project Description

The Student Study Portal is a web-based application designed to help students manage their study resources and daily academic activities in one place.
It provides useful tools such as books access, dictionary search, homework management, notes, to-do lists, educational videos, and more.

The main goal of this project is to make learning easier by combining multiple study tools in a single platform.

---

🚀 Features

🔐 Authentication

- Register Page – New users can create an account.
- Login Page – Registered users can log in securely.
- Logout Page – Users can safely log out of the portal.

👤 User Profile

- Profile Page – Displays student information and account details.

📖 Study Resources

- Books Page – Students can access different books for study.
- Dictionary – Search for word meanings quickly.
- Wikipedia Integration – Allows students to search educational topics directly.

📝 Study Management

- Homework Page – Students can add and track homework tasks.
- Notes Page – Students can create and save personal study notes.
- To-Do Page – Helps students organize daily study tasks.

🎥 Learning Tools

- YouTube Integration – Students can search and watch educational videos.

📏 Unit Converter

- Conversion Tools – Convert units such as:
  - Mass
  - Length
  - Other measurement units
